# brand_assets/

Drop the client's real brand assets here **before** designing. The
`frontend-design` workflow checks this folder first and uses whatever it finds
instead of placeholders.

Put here (if available):
- **Logos** — SVG preferred, PNG ok (light + dark variants)
- **Color guide** — exact hex values / palette
- **Fonts** — or the names of the brand typefaces
- **Style guide** — any PDF/image with spacing, voice, do/don'ts
- **Photography** — real imagery to use in place of `placehold.co`

If a logo or palette exists here, use those exact values — do not invent brand
colors or swap in placeholders where real assets are available.
